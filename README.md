# 🛡️ Master Hacker Journey: 90 Days to Mastery

### **Project Overview**
This repository serves as my technical laboratory and documentation hub for a 12-week intensive cybersecurity curriculum based on *"How to Become a Master Hacker"* by Occupy The Web. My goal is to transition from foundational security concepts to advanced offensive and defensive mastery.

---

### **🛠️ The Lab Environment**
*   **Primary Attacker:** Kali Linux (Rolling Release)
*   **Virtualization:** VirtualBox / VMware Workstation
*   **Target VMs:** Metasploitable 2, Windows 10 (Evaluation), and custom VulnHub machines.
*   **Defensive Focus:** Analyzing attack signatures for SOC operations.

---

### **📅 12-Week Roadmap**

| Phase | Timeline | Focus Area |
| :--- | :--- | :--- |
| **Phase 1** | Weeks 1-3 | Advanced Reconnaissance & OSINT |
| **Phase 2** | Weeks 4-6 | Exploitation & Password Cracking |
| **Phase 3** | Weeks 7-9 | Web Application & Network Attacks |
| **Phase 4** | Weeks 10-12 | Evasion, Scripting, and Post-Exploitation |

---

### **📑 Repository Structure**
*   **`/Weekly-Logs`**: Detailed write-ups and command history for each lab.
*   **`/Scripts`**: Custom Python and Bash automation tools developed during the series.
*   **`/Reports`**: Professional-grade vulnerability triage reports and executive summaries.

### **⚖️ Disclaimer**
All activities documented in this repository are performed in a controlled, isolated laboratory environment for educational purposes only. Unauthorized access to computer systems is illegal and unethical. No actions performed here are intended for use on systems without explicit, written consent.
